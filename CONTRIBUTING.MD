# Contributing Workflow

1. Pull latest dev: `git checkout dev && git pull`
2. Create a feature branch: `git checkout -b feature/short-name`
3. Commit with clear messages: `git commit -m "feat: add booking form validation"`
4. Push: `git push -u origin feature/short-name`
5. Open a PR into `dev` on GitHub
6. Wait for review/approval before merging
7. Delete the branch after merge
